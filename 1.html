O que é JavaScript?

A motivação para a criação do JavaScript: O JavaScript foi criado em 1995 com o objetivo de tornar as páginas da web mais dinâmicas e interativas. Ele permite que ações sejam realizadas diretamente no navegador, como responder a cliques e mudar elementos da página sem precisar recarregá-la completamente.

A diferença entre execução no navegador e no servidor: No navegador (lado do cliente), o JavaScript é usado para interagir com o usuário e modificar a interface da página em tempo real. No servidor, com tecnologias como Node.js, ele é usado para processar dados, acessar bancos de dados e responder a requisições. A principal diferença é que no navegador o foco é na experiência do usuário, enquanto no servidor é na lógica de funcionamento do sistema.

O conceito de sandbox no navegador: Quando o JavaScript é executado no navegador, ele roda dentro de uma "sandbox", que é um ambiente isolado e seguro. Isso impede que ele tenha acesso direto ao sistema do usuário, protegendo contra possíveis ações maliciosas.

Objetos globais como document, console e alert: Durante a execução no navegador, o JavaScript tem acesso a alguns objetos globais importantes:
document: permite acessar e modificar o conteúdo da página HTML.
console: usado para exibir mensagens no console do navegador, útil para testes e depuração.
alert: exibe uma caixa de mensagem na tela para o usuário.
